ئەم فایلە «نەخشەی تەواوی» پرۆژەکەیە — بە زمانی کوردی و بە وردی تەکنیکی.
دەتوانیت تەواوی بخەیتە ناو Gemini (یان Claude، ChatGPT، Cursor…) بۆ:

  · تێگەیشتنی ئارکیتێکتچەر و پێکهاتەی کۆد
  · چاککردنی باگ یان زیادکردنی فیچەری نوێ
  · ڕێنمایی بۆ گۆڕانکاری لە POS، پسوولە، داشبۆرد، API، یان deploy

  ناوی پرۆژە لە npm ................. m-and-m
  ناوی بەرهەم لە ڕووکار .............. Hawre System
  فۆڵدەری ڕەگە (Windows) ............ m&&m  (تێبینی: & لە ڕێڕەودا کێشەی shell دروست دەکات)

  بەشی ١ — پوختەی سیستەم: چییە و بۆ کێیە؟

Hawre System بەرنامەیەکی وێبی پیشەییە بۆ دووکانەکانی مۆبایل و پێداویستی —
بۆ فرۆشتنی ڕۆژانە، کۆگا، کڕین، قەرز، قاسە و ڕاپۆرتی دارایی. هەموو شتێک لە
یەک شوێندا کۆدەکراوە: لە کاشێرەوە تا چاپی پسوولە و داشبۆردی بەڕێوەبەر.

  ┌─ بەشی کارکردن ─────────────────────────────────────────────────────────┐
  │                                                                          │
  │  فرۆشتن (POS)     کڕیار، بەرهەم، چەند نرخ بۆ هەمان کاڵا، چاپی پسوولە    │
  │  کۆگا             بەرهەم، وێنە، SKU، بارکۆد، ئاگاداری کەمی کۆگا          │
  │  کڕین             لە کۆمپانیا، USD/IQD، قەرز یان نەقد، گەڕاندنەوە       │
  │  قەرز             کڕیار، دابینکەر، کارمەند                              │
  │  دارایی           خەرجی، هاوبەش، قازانج، قاسەی بچووک، کاشێر             │
  │  ڕاپۆرت           جەرد (Jard)، قازانج، داشبۆرد بە ماوەی بەروار            │
  │  ڕێکخستن         نرخی دۆلار، شێوەی پسوولە، QR، ڕەنگ و تێم               │
  │  گشتی            لاندینگی QR بۆ سۆشیاڵ میدیا · چەند دووکان (multi-shop)  │
  │                                                                          │
  └──────────────────────────────────────────────────────────────────────────┘

  پێکهاتەی تەکنیکی (یەک ڕستە):

      React  +  TypeScript  +  Vite  +  Tailwind
              ↓  HTTP / JSON  ↓
      Django  +  Django REST Framework  +  PostgreSQL (لە production)

  چوونەژوورەوە: ئیمەیڵ + تێپەڕەوش · HTTP Basic Auth · GET /api/users/me/
  جیاکردنەوەی داتا: هەر دووکان = یەک «Shop» (tenant) · داتاکانی دووکانەکان تێکەڵ نابن

  ┌─ بەرهەمهێنان (Production — نموونە Linode) ───────────────────────────────┐
  │                                                                          │
  │   Nginx  ──►  frontend/dist (فایلە ستاتیکەکان)                           │

  │                                                                          │
  └──────────────────────────────────────────────────────────────────────────┘

  ناسنامە (Authentication)
  ─────────────────────────
  · DRF: SessionAuthentication + BasicAuthentication
  · فرۆنتێند: setBasicAuth(email, password) لە lib/api.ts
  · دوای لۆگین: GET /api/users/me/ → role، shop، permissions
  · هەموو داواکاری: Authorization: Basic … + credentials: include

  چەند دووکان (Multi-tenant)
  ───────────────────────────
  · هەر تۆمارێکی بازرگانی shop_id هەیە
  · بەکارهێنەری ئاسایی تەنها داتای دووکانی خۆی دەبینێت
  · Superuser: هەڵبژاردنی دووکان → ?shop_id= لە API
  · Global View: بێ shop_id بۆ بینینی گشتی

  داتابەیس
  ────────
  · Dev: SQLite (backend/db.sqlite3) ئەگەر DJANGO_USE_SQLITE=true
  · Production: PostgreSQL
  · کاتژمێر: DJANGO_BUSINESS_TZ=Asia/Baghdad (بۆ فلتەری بەروار)

  تۆڕی نەماو (Offline)
  ─────────────────────
  · داواکارییەکانی گۆڕان لە نۆرە دەخرێن · دوای گەڕانەوەی ئینتەرنێت sync دەبن

  بەشی ٨ — فرۆشتن (POS) و پسوولە — وردەکاری
════════════════════════════════════════════════════════════════════════════════

  POS — PosPage.tsx
  ─────────────────
  · گەڕانی خێرای کڕیار و بەرهەم
  · سەبەتە: هەر هێڵ lineId ـی تایبەتی هەیە → هەمان بەرهەم چەند جار بە نرخی جیاواز
    (نموونە: جوملە بۆ برادەرێک و تاک بۆ کڕیارێکی تر)
  · بەرهەمی دەستی لە گەڕان
  · پارەی وەرگیراو بە USD و IQD (پەیوەندی نرخی دۆلار)
  · پاشەکەوتکردنی draft · دەستکاری فرۆشتن (?edit=) · گەڕاندنەوە (?return_sale=)

  ڕێڕەوی Tab (کیبۆرد):
  ─────────────────────
  کڕیار → بەرهەم → دانە → نرخ → دووبارە بەرهەم (بۆ کاڵای داهاتوو)
  کلیک لە پارەی وەرگیراو → Tab لە نێوان خانەکانی پارەدان

  پسوولە — receiptHtml.ts
  ───────────────────────
  · HTML لە براوزەر دروست دەبێت · چاپی A4
  · QR لە پسوولە (لە کلایەنت دروست دەبێت)
  · نرخی گۆڕان (IQD/100 USD) لە چاپدا نیشان نادرێت
  · ناوی دوکان لە سەرەوە نییە — لەسەر «واژووی بەرپرسی فرۆشتن» دەردەکەوێت
    (لە ڕێکخستن: shop_name_ku · shop_name_en)

  ────────────────────
  ١) cd backend && python -m venv venv && pip install -r requirements.txt
